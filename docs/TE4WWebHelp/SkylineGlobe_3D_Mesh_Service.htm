SkylineGlobe 3D Mesh Service

Powerful 3D city model service technology for streaming 3D Mesh Layer (3DML) databases from 3DML files to remote TerraExplorer clients. 3D mesh layers (3DML) are high-resolution, textured, 3D mesh models that have been compressed and stream-optimized to accelerate loading speed and efficiently handle memory. The 3D Mesh Service also exposes the data as 3D tiles for TerraExplorer for Web and Cesium based clients.
