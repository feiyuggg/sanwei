Opening TerraExplorer for Web

To open TerraExplorer for Web:

§  Type the TE4W URL in your browser.

Note:        Your URL can specify a specific server site, specific TE4W configuration, or TE4W project to load, e.g., http://www.skylineglobe.com/sg/SiteOne/terraexplorerweb/terraexplorer.html? config=config.kml&catalogid=12345. See “SGS User Manual” for more information.
