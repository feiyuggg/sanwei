TE4W Sidebar Tabs

All of TerraExplorer for Web’s commands are available from TE4W’s sidebar:

§  Navigate tab  – Select a navigation mode, activate underground mode, fly around an object or to a saved location, or play a presentation

§  Analysis tab  – Use one of the following analysis tools: Query, Distance, Horizontal Area, Profile, Shadow, Contour Map, Slope Map and Viewshed.

§  Layers tab  – Fly to and manage the project’s layers or load a new layer from SkylineGlobe Server.

§  Settings  – View basic information about the application version and SkylineGlobe Server, and set the screen overlay and units.
