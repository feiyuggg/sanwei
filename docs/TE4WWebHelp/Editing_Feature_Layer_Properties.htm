Editing Feature Layer Properties

Basic layer and feature properties can be modified from the layer’s property sheet. 

To edit a layer’s properties:

1.     On the sidebar, click the Layers tab.

2.     Click the layer you want to edit, and then click Edit   > Layer Properties . The Layer’s property sheet is displayed.

3.     Edit the layer properties as required. The specific properties displayed depend on the feature layer type.

Property

Description

General

Name

Name of layer.

Type

Feature type: point, polyline, or polygon (Read-only).

Appearance

Note:    The display options available depend on the layer type.

Display Feature Point As

Select whether to represent each point feature with an icon or label.

Color

Color of the features (polylines and polygons). Click the color bar to open the Color dialog box, or type the color code in hexadecimal 0xBBGGRR format (B = Blue channel 00-ff, G = Green channel 00-ff, R = Red channel 00-ff).

Opacity

Opacity for layer’s outline (0-1) (polygons).

Outline Color

Outline color for each of the layer’s features (polygons).

Label Text

For point layers represented by a label (i.e., label is selected in “Display Feature Point As”), the label text. Click Apply Classification to select a field attribute on which to base the text label content.

Icon URL

For point layers represented by an icon (i.e., icon is selected in “Display Feature Point As”), URL to the icon file.

Line Width

Line width in pixels (1-5) (polylines).

Note:    When layers published in TerraExplorer Pro are loaded into your project: all meter values are displayed as 1 pixel wide.

Position

Altitude Method

Sets the altitude method to be used by the layer:

§  Select Relative (to Terrain) to place each point of the layer at a specified altitude above the ground, defined by its Altitude Offset.

§  Select Absolute to place each point at a specified altitude (defined by the Altitude Offset) above the terrain database vertical datum base ellipsoid. You can use this feature to create layers with known absolute altitude values that are not dependent on the terrain elevation values.

§  Select On Terrain to clamp the layer to the terrain or model.

Altitude Offset

Specified altitude above the ground to place each point of the layer. This property is only displayed if “Relative” or “Absolute” were selected as the Altitude Method.

Clamp To

The surface to which to clamp the layer: Model or Terrain. This property is only displayed if “On Terrain” was selected as the Altitude Method.

 

4.     Click OK.
