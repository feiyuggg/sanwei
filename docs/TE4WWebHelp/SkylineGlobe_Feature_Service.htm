SkylineGlobe Feature Service

Powerful feature service technology for streaming point, polyline, and polygon feature layers from files and from the Oracle, SQL server, PostGIS and ArcSDE databases to remote TerraExplorer clients and any application that reads the standard OGC WFS protocol. The Feature Service can stream most TerraExplorer supported feature layer formats. It also supports advanced spatial and attribute queries from TerraExplorer and other WFS clients.
