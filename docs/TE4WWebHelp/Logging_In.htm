Logging In

To log in:

1.     On the top of the sidebar, click the Log in link. The SkylineGlobe Login dialog box is displayed.

2.     Type your User Name and Password.

3.     If you want your login details to be automatically completed the next time you log in, select Remember me next time.

4.     Click Log in. A message is displayed informing you that some additional layers may be available for the new login. Click the link to refresh the layers list by reloading TerraExplorer for Web.
