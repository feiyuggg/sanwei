Terrain Profile

The Terrain Profile tool displays the terrain elevation profile along a defined path, and related information on this profile such as maximum and minimum elevation values and slope.

To use the Terrain Profile tool:

1.     On the sidebar, click the Analysis tab > Profile.

2.     Click any point on the terrain to define the start point of the measurement. Click to set the second point and then again for each additional segment.

3.     Click Terrain Profile  to calculate the terrain profile. When calculation is complete, a terrain profile graph is displayed, showing the terrain elevation profile.

4.     Select the check boxes of the information you want indicated on the graph: polyline waypoints, min/max altitude and max slope.

5.     Point to any point on the graph to display its X and Y coordinates, height and slope. If you want to display the values and also jump to the point on the terrain, then click instead of pointing to it.

6.     Click Delete Point  to delete the last point or Delete All  to delete all points.
