Adding a Favorite Location

You can save favorite locations in “Locations and Presentations” (available from the sidebar’s Navigate tab).

To add a location to “Locations and Presentations”:

1.     Navigate to the location. See “Navigating in the 3D World” for more information.

2.     On the sidebar, click the Navigate tab, and then in the Tools section, click Create Location. The My Locations dialog opens

3.     Type a Name for this location. You can also type a Message to display when the layer is clicked in the 3D Window, and a star color that will represent the location.
