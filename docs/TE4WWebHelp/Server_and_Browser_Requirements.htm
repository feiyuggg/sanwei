Server and Browser Requirements

Operating System

Windows / Linux / MacOS / Android / iOS

System Memory

2GB of RAM (4 GB or more recommended)

Processor

4 cores (8 cores recommended)

Browser

Chrome, Firefox, Safari (Only supported in Android and Mac), Opera, Edge

 

Note:    TE4W does not support on-board video cards.

*Internet Explorer 11 Limitations

TE4W 7.1 supports Internet Explorer 11 with the limitations listed below. It is recommended to use one of the other supported browsers for full support.

§  3D mesh layer ground surface is not supported

§  When a point on a 3DML is selected for rotation, TE4W rotates around the terrain behind the point due to IE graphics limitations

§  Polygon feature layers do not render immediately to the correct altitude above the terrain

§  While using the Query tool, the Query location icon stops displaying when zooming in

§  High memory usage
