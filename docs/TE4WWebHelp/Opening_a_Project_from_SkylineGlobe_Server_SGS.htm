Opening a Project from SkylineGlobe Server (SGS)

See “User Login” in this chapter for information on logging in to your own SGS account.

To open a new project from SGS:

1.     On the sidebar, click Settings  > Open Another Project. The SkylineGlobe Layers dialog box is displayed showing all TE4W projects on your SkylineGlobe server.

2.     Search for the required project. See “Searching for SkylineGlobe Layers” in the “Managing Layers” chapter for information.

3.     Select the required project and click Load.
