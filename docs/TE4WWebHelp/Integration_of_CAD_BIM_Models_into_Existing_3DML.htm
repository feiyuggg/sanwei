Modify 3D Mesh Layer

TE4W 7.2.1 enables you to integrate CAD/BIM models into an existing city-wide 3D mesh layer, by using the Modify Terrain object to flatten 3D mesh layers in a polygonal area, and then insert the models in their correct positions. Modify Terrain objects can be created in TerraExplorer Pro and then published to SGS or created via the TE4W API.
