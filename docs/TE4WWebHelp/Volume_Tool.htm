Volume Tool

The Volume Analysis tool analyzes the volume of the terrain or model removed or added by drawn horizontal planes. The plane is marked by drawing a polygon in the 3D Window.

To analyze the volume of removed or added terrain or model:

1.     On the sidebar, click the Analysis tab > Volume.

2.     With the cursor in the 3D Window, click to add the first point of the horizontal plane polygon.

3.     Click again to add additional points of the polygon until you have marked the entire shape. You must place at least three points to define the polygon. Right-click or click the check box  to finish marking the polygon. Before right-clicking, you can click Delete Point  to delete the last point.

4.     Click Volume Analysis Properties  to set its properties, and then click OK:

Object Parameter

Description

Name

Name of the volume analysis object as it will be displayed in the sidebar.

Quality

Precision of the volume analysis calculation.

5.     Click Calculate . The Volume Analysis dialog box displays the removed and added volume, in addition to the area and perimeter measurements.

6.     If you want to delete the volume analysis polygon, click Delete .
